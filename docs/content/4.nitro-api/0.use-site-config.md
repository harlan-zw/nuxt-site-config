---
title: getSiteConfig()
description: How to use Site Config within Nitro.
---

Access the current site config within a Nitro event handler. Requires an `H3Event` as the first parameter.

::warning
v4 removed the server helper `useSiteConfig(event)`{lang="ts"} to resolve an auto-import name collision. Use `getSiteConfig(event)`{lang="ts"} instead. Note that `useSiteConfig()`{lang="ts"} without an event parameter is still available in Nuxt app code (client and SSR).
::

## Usage

```ts [serverMiddleware.ts]
import { defineEventHandler } from '#imports'
import { getSiteConfig } from '#site-config/server/composables'

export default defineEventHandler((e) => {
  const siteConfig = getSiteConfig(e)
  console.log(siteConfig.name)
})
```
